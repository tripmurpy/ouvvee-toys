# Tahap 3: Pemilihan Sistem Manajemen Basis Data

DBMS yang digunakan adalah **MySQL** dengan model data relasional.

## Alasan Pemilihan MySQL

| Alasan | Keterangan |
|---|---|
| Mendukung model relasional | Data dapat dibuat dalam bentuk tabel yang saling berhubungan. |
| Mendukung primary key dan foreign key | Cocok untuk menjaga hubungan data pesanan, barang, pembayaran, dan pengiriman. |
| Cocok untuk sistem penjualan online | MySQL umum digunakan pada website e-commerce. |
| Mudah digunakan | Query SQL mudah dipahami dan digunakan untuk laporan. |
| Mendukung normalisasi | Data dapat dipisah agar tidak terjadi duplikasi yang tidak perlu. |

## Kesimpulan

Sistem Penjualan Barang Ouvvee menggunakan **MySQL** karena kebutuhan sistem berbentuk data relasional seperti user, barang, pesanan, pembayaran, dan pengiriman.
